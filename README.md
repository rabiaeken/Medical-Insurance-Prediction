# Medical Insurance Cost Prediction

This project aims to predict annual medical insurance costs using machine learning techniques. It covers the entire data science pipeline, including data cleaning, exploratory data analysis (EDA), feature engineering, and model evaluation.

## 🚀 Key Features
* **Data Preprocessing:** Handling missing values, outlier detection using the IQR method, and feature scaling with RobustScaler.
* **Exploratory Data Analysis:** Detailed visualization of cost correlations with age, BMI, and smoking habits.
* **Machine Learning Models:** Performance comparison of three different algorithms:
    * Linear Regression
    * Random Forest Regressor
    * K-Nearest Neighbors (KNN)
* **Feature Engineering:** Creation of new metrics such as `healthcare_utilization`, `chronic_disease_count`, and `bmi_risk` groups.

## 📊 Results
The models were evaluated using MAE, RMSE, and R² scores. The **Random Forest Regressor** achieved the highest accuracy in predicting insurance costs.
