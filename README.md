# Medical-Insurance-Prediction
Machine learning project to predict medical insurance costs using Linear Regression, Random Forest, and KNN algorithms.
---
## 👥 Proje Ekibi
Bu çalışma, **Rabia Eken** ve **Çağla Özen** tarafından ortak bir proje olarak geliştirilmiştir. Veri seti analizi, özellik mühendisliği ve KNN, Random Forest, Linear Regression modellerinin kurulması süreçleri birlikte yürütülmüştür.
